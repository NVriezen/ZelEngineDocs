Differences Between Engines
=================================
This may not be the first game engine you use.
However the Zel Game Engine does things a bit differently.
Therefore the `Migration Guide <https://nvriezen.github.io/ZelMigrationGuide/>`_ will help you get familiar with the Zel Game Engine.
It explains the differences between popular game engines you might have used before.

It even contains explanations on why this engine does things in a different way.
Highlighting some of the advantages this other manner of thinking brings with it.
